### Installation

run:

```
npm install codemirror
```

use:

```js
import '@vonage/vivid/button';
```

```html preview
<vwc-button label="Click me"></vwc-button>
```

See the [open issues](https://github.com/vonage/vivid-3/issues) for a full list of proposed features (and known issues).
