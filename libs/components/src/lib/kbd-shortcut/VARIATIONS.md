## Two Keys

A common shortcut with two keys.

```html preview inline
<vwc-kbd-shortcut>
	<vwc-kbd-key name="Ctrl"></vwc-kbd-key>
	<vwc-kbd-key name="C"></vwc-kbd-key>
</vwc-kbd-shortcut>
```

## Three Keys

A shortcut with three keys.

```html preview inline
<vwc-kbd-shortcut>
	<vwc-kbd-key name="Ctrl"></vwc-kbd-key>
	<vwc-kbd-key name="Shift"></vwc-kbd-key>
	<vwc-kbd-key name="P"></vwc-kbd-key>
</vwc-kbd-shortcut>
```

## Mac-Style Shortcut

Using the Cmd key for macOS shortcuts.

```html preview inline
<vwc-kbd-shortcut>
	<vwc-kbd-key name="Cmd"></vwc-kbd-key>
	<vwc-kbd-key name="Shift"></vwc-kbd-key>
	<vwc-kbd-key name="Z"></vwc-kbd-key>
</vwc-kbd-shortcut>
```

## Single Key

A single key can also be wrapped in a Kbd Shortcut.

```html preview inline
<vwc-kbd-shortcut>
	<vwc-kbd-key name="Enter"></vwc-kbd-key>
</vwc-kbd-shortcut>
```
