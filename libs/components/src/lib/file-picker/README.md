# File picker

Represents a file picker custom element.

```js
<script type="module">
    import '@vonage/vivid/file-picker';
</script>
```

## Members

### Text

Add a `text` attribute to add text to the file picker.

- Type: `string`
- Default: `undefined`

```html preview
<vwc-file-picker text="Drag & Drop files here"></vwc-file-picker>
```
