# Slider

Represents a slider custom element.

```js
<script type="module">import '@vonage/vivid/slider';</script>
```

```html preview
<vwc-slider></vwc-slider>
```

## Members

### Orientation

- Type: `'horizontal'` | `'vertical'`
- Default: `horizontal`

```html preview blocks
 <vwc-slider orientation="vertical"></vwc-slider>
```
