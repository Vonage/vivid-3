# Country Indicator Guidelines

Use the Country Indicator to show a country's flag and country code together. Set the **code** attribute (ISO 3166-1 alpha-2, e.g. "GB", "UK", "US"); the component shows the matching flag (from the [flag-icons](https://github.com/lipis/flag-icons) library) and the country code (e.g. "UK", "US") automatically. Use the **icon** slot only when you need a custom flag or graphic instead of the default.
