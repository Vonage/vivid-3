## Size

Icon size by default is inline.
But can be set to predefined size

```html preview
<vwc-icon type="home" size="small"></vwc-icon>
<vwc-icon type="home" size="medium"></vwc-icon>
<vwc-icon type="home" size="large"></vwc-icon>
```
